NalCSS

What is NalCSS?

NalCSS is a vanilla CSS created by @123jimin for styling simple HTML pages.
For example, here are a few use cases for NalCSS:

  • Styling hand-written HTML pages (like this one!), without using any frameworks.
  • Styling a simple React app where using UI libraries can be overkill.

Please note that NalCSS is currently work-in-progress.

Also, please note that NalCSS does not aim to be a competent CSS framework. Come up with your own CSS instead of using NalCSS if you need a more complex design.

Features of NalCSS

  • Pure CSS, without any JavaScript.
  • Supports most modern web browsers.
  • Responsible design; works well on both desktop and mobile.
  • Does not require any build tools.
  • Basic customization with CSS variables.
  • No external dependencies.
    • NalCSS does not make use of any external resources, such as fonts or icons.
    • Technically, reset.css is a dependency, but it can easily be added to your project.
